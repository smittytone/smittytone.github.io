MNU

A macOS menu bar utility

Latest Release: 1.0.0

 

About MNU

MNU is a macOS main menu bar utility which provides not only a number of handy system controls, but also allows you to invoke shell scripts that you use frequently.

MNU comes pre-configured with a number of useful items:

  • A switch to enable or disable macOS’s Dark Mode (Mojave and up)
  • A switch to enable or disable macOS’ Desktop
  • A trigger to update your Git repos (Note requires gitup)
  • A trigger to update your Homebrew installation (brew update)
  • A trigger to upgrade your Homebrew installation (brew upgrade)

Any or all of these stock items can be hidden from view if you don’t need them, and you can add your own items. You can re-order the items too.

Using MNU For the First Time

When you first use one of MNU’s items, macOS will request permission to grant either System Events and/or Terminal, depending on which item you select. Please OK this request, which grants MNU the right to, respectively, enable/disable Dark Mode, and run the commands that are part of menu items in the Terminal.

Important MNU will not fully function without these permissions.

You can check the permissions granted to MNU by opening System Preferences > Security & Privacy > Privacy > Automation.

How To Configure MNU

To configure MNU, click on the gearwheel icon in the last item on the menu:


 
MNU’s controls (left to right): Quit, Configure and Help

This pops up the Configure MNU window: make sure the Menu Items tab is showing.

The Menu Items tab lists the menu items currently set up for MNU in the order in which they appear in the menu itself. You can reorder the items just by grabbing an item and dragging it to the location you prefer. When you’re done, click Save to update MNU, or Cancel to ignore any changes you have made.

The pre-installed menu items all feature a button which allows you to hide any such items that you don’t need. If you later decide you’d like to see a hidden item after, just click the button (now marked Show) to bring it back to MNU.

How To Add And Edit Your Own Menu Items

Initially, MNU contains only the stock items, but you can add further ones by clicking on Add Item. This presents a sheet in which you can enter a Terminal command that can then be invoked from MNU’s main menu. The command can be a simple request to change directory (for example, cd $HOME), or can be a complex command line sequence that invokes a script or command and passes in fixed arguments (such as, ./myimageresizescript.py -size 640 480).

You will also need to give the command a name by which it will be listed in the MNU menu. This should not match that of an existing item. When you are done, click Add to add the item to MNU, or Cancel to discard your changes.

The new item will now be listed by the Menu Items tab, but it won‘t be added to MNU until you click Save; clicking Cancel will cause the new item to be discarded.

Items you have added can be deleted from the list (and thus MNU’s menu), or they can be edited if you need to make a change: just click on the Delete or Edit, respectively.

When MNU quits — you have restarted or shutdown your Mac, or you have clicked the quit icon at the bottom of the menu — it saves the current state of the menu so it’s ready for next time.
