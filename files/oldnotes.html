Squinter 2.1.125 (26 October 2018)

Improvements

  • Begin using Electric Imp as the source of Electric Imp library version numbers.
  • Better logging of updated included libraries and files in source code.
  • When a shared Product is selected, the Shared Products menu item is marked to aid visual navigation.
  • Make Inspector link button graphics work better with Mojave Dark Mode.

Bug Fixes

  • A shared Product won’t be selected at login if there is an owned Product available to be selected.
  • Halting a device’s log stream no longer causes other devices’ streams to change colour.
  • Switching accounts no longer causes the Products of the account you’ve just logged into to be displayed as shared Products.
  • Squinter will now correctly begin watching a Device Group’s new source code file(s) when the existing one(s) are replaced (and stop watching the old files).
  • Correctly mark projects with updated imported libraries and files as in need of saving.

Squinter 2.0.121 Beta 14 (15 June 2018)

Improvements

  • Break out list of products on which you are collaborating.
    • Shared products are listed by owner account.
  • Support file versioning.
    • Use #version "x.y.z" to indicate a file’s version.
    • As before class versions should entered as static version = "x.y.z" or
      const version = "x.y.z" within the class definition.
  • Add sign-in support for multiple impClouds, ie. AWS and Azure.
  • Improve account switching messaging and account labelling.
  • Significantly speed up writing text to the log.
  • Present Squinter's own log message tags: Device, Agent, Server.
  • Improve EI library checking and reporting.
  • Add clearer warnings when libraries and/or files have been removed prior to compilation.
  • Add support for eight simultaneous logging devices (from five).
  • Add preference to keep devices' status updated.
  • Improve commit list display.
  • Kill all connections when the Mac is about to sleep.

Bug Fixes

  • Fix Preferences panel's bold text checkbox operation.
  • Fix placement of Open and Save panels' accessory views.
  • Fix appearance of new Device Groups in the Project Inspector.
